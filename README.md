# 일일업무보고서 앱

간편하고 효율적인 일일업무보고서 작성 및 관리 애플리케이션입니다.

## 주요 기능

### 📝 보고서 작성

- **기본 정보**: 보고서 날짜, 부서명, 작성자
- **업무 내용**: 업무 요약, 완료된 업무, 진행 중인 업무
- **기타 정보**: 이슈 및 문제점, 내일 계획, 기타 사항

### 💾 데이터 관리

- **저장**: 로컬 스토리지를 이용한 보고서 저장
- **불러오기**: 저장된 보고서 목록에서 선택하여 불러오기
- **초기화**: 현재 작성 중인 폼 초기화
- **삭제**: 저장된 보고서 삭제

### 📊 내보내기

- **Excel 내보내기**: .xlsx 형식으로 보고서 내보내기
- **PDF 내보내기**: .pdf 형식으로 보고서 내보내기

## 사용 방법

### 1. 보고서 작성

1. 웹 브라우저에서 `index.html` 파일을 열어주세요
2. 보고서 날짜, 부서명, 작성자를 입력하세요 (필수 항목)
3. 각 섹션에 해당하는 내용을 작성하세요
4. **저장** 버튼을 클릭하여 보고서를 저장하세요

### 2. 보고서 관리

- **불러오기**: 저장된 보고서 목록에서 원하는 보고서를 선택하여 불러오기
- **초기화**: 현재 작성 중인 내용을 모두 지우고 새로 시작
- **삭제**: 저장된 보고서를 영구적으로 삭제

### 3. 내보내기

- **Excel 내보내기**: 현재 작성된 보고서를 Excel 파일로 다운로드
- **PDF 내보내기**: 현재 작성된 보고서를 PDF 파일로 다운로드

## 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: 반응형 디자인, 그라데이션, 애니메이션
- **JavaScript (ES6+)**: 클래스 기반 모듈화, 로컬 스토리지 API
- **외부 라이브러리**:
  - SheetJS (xlsx): Excel 파일 생성
  - jsPDF: PDF 파일 생성

## 브라우저 지원

- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+

## 특징

### 🎨 사용자 친화적 인터페이스

- 모던하고 직관적인 디자인
- 반응형 레이아웃으로 모바일/태블릿 지원
- 실시간 알림 시스템

### 🔒 데이터 보안

- 모든 데이터는 로컬 브라우저에만 저장
- 외부 서버로 데이터 전송 없음
- 개인정보 보호

### ⚡ 빠른 성능

- 클라이언트 사이드에서 모든 작업 처리
- 로컬 스토리지 활용으로 빠른 데이터 접근
- CDN을 통한 외부 라이브러리 로딩

## 주의사항

- 브라우저의 로컬 스토리지를 사용하므로, 브라우저 데이터를 삭제하면 저장된 보고서도 함께 삭제됩니다
- 다른 컴퓨터나 브라우저에서는 저장된 데이터에 접근할 수 없습니다
- 중요한 데이터는 정기적으로 Excel이나 PDF로 내보내기하여 백업하시기 바랍니다
